# VofSwords

Personal portfolio website

---

## Requirements

> **Node.js** 16.x  
> **Yarn** 1.x

## Install

---

### **yarn**

```
$ git clone https://github.com/VofSwords/vofswords.page.git
$ cd vofswords.page
$ yarn 
```

## Scripts

---

### start

Builds app and launches it

### dev

Launches front-end development server

### build

Builds front-end and back-end for production

### build-web

Builds front-end for production

### dev-server

Launches back-end for development

### build-server

Builds back-end for production

### start-server

Launches the app
